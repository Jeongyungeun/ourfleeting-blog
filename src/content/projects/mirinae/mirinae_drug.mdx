# 프로젝트 설명(PRD)
이 프로젝트는 환자가 처방전 사진을 찍어 앱에 로드하고 약국을 선택하면 처방전만 미리 보내 약을 먼저 짓고 환자가 시간 될 때 찾아가는 시스템을 간편하게 해주는 모바일 앱입니다.

# 기술 스택
- Fontend : Flutter
- Backend : Rust(Actix-web)
- Dashboard : Rust(Yew)
- DB : Postgresql
- Redis
- deploy : fly.io

# 사용 Flow
- 환자가 처방전을 받는다.
- 사진을 찍어 원하는 약국에 전송한다.(현재는 내약국만 등록 되어있다.)
- 약국에서 약을 조제하고 모든 조제가 완료되면 환자에게 푸쉬알람을 보낸다.
- 환자가 약을 수령한다.

# 전달 하고자 하는 핵심 가치.
- 자기가 원하는 시간에 기다림 없이 약을 수령할 수 있다.
- 약을 찾으러 여러 약국에 전화 또는 찾아 다닐 필요가 없다.

# 발전 방향
- 동네 약국과 환자를 이어주는 onDemand 플랫폼이 되고자 한다.
- 처방전 배달 또는 일반의약품 배달 시장의 전초 프로젝트
